JML Usage Examples

Basic Syntax

$div(
	$h1({style: 'color: #c88100;'}, 'Hello, World!'),
	$p('Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.')
)
$div( $h1({style: 'color: #c88100;'}, 'Hello, World!'), $p('Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.') )

Great Live Update Feature

$div(
	$input({
		jmlModal: 'myModal',
		type: 'text',
		placeholder: "Type here to see JML's live update feature",
		// value: 'Alter this text',
		style: 'width: 275px; padding: 10px; border: 1px solid lightblue; border-radius: 2px;'
	}),
	$br(),
	$h3({jmlBind: 'myModal', style: 'color: #286bcf; text-transform: uppercase;'})
)
$div( $input({ jmlModal: 'myModal', type: 'text', placeholder: "Type here to see JML's live update feature", // value: 'Alter this text', style: 'width: 275px; padding: 10px; border: 1px solid lightblue; border-radius: 2px;' }), $br(), $h3({jmlBind: 'myModal', style: 'color: #286bcf; text-transform: uppercase;'}, 'Hi, World!') )

JML Logical Elements

The IF element

$div(
	$if(
		true,
		$h1('The IF element is working!'),
		$h3('The IF element is working!')
	)
)
$div( $if(true, $h1('The IF element is working!'), $h3('The IF element is working!')) )

The IF element

$div(
	$if(
		true,
		$h1({style: 'color: red;'},'The IF element is working!'),
		$h1({style: 'color: blue;'},'The IF element is working!')
	)
)
$div( $if( true, $h1({style: 'color: red;'},'The IF element is working!'), $h1({style: 'color: blue;'},'The IF element is working!') ) )
